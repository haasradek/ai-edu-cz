# Web — Astro + Vercel

Tato složka bude obsahovat zdrojový kód webu (Fáze 2).

## Stack

- **Framework:** Astro (statický generátor, rychlý, SEO-friendly)
- **Hosting:** Vercel (free tier, automatický deploy z GitHubu)
- **Styling:** Tailwind CSS

## Inicializace projektu (až budeme připravení)

```bash
cd web
npm create astro@latest .
npm install @astrojs/tailwind tailwindcss
```

## Deploy na Vercel

1. Push na GitHub (automaticky)
2. Vercel detekuje změny a nasadí web
3. URL: nastav v `.env` → `PROJECT_URL`
