# Assets — Obrázky, loga, vizuály

Tato složka obsahuje statické soubory projektu.

## Struktura

```
assets/
├── logo/          — Logo AI-EDU-CZ (různé formáty)
├── thumbnails/    — YouTube thumbnaily
├── instagram/     — Obrázky pro Instagram posty
├── infografiky/   — Vzdělávací infografiky pro lekce
└── brand/         — Barvy, fonty, brand guidelines
```

## Brand barvy (dashboard paleta)

| Název | HEX | Použití |
|-------|-----|---------|
| Cyan | `#00d4ff` | Primární akcent |
| Purple | `#7c3aed` | Sekundární akcent |
| Green | `#00ff88` | Úspěch, hotovo |
| Red | `#ff3355` | Chyba, varování |
| Background | `#000000` | Hlavní pozadí |

## Nástroje pro tvorbu vizuálů

- **Canva** — infografiky, Instagram carousel, thumbnaily
- **DALL-E / Leonardo.ai** — AI generované obrázky
- **Gamma** — prezentace a slidedeck
